	This web-application describes the use of a Database Management System that interacts with 
the end users and a database for a pharmacy(medicine store). The end users here are customers,
employees and the owner of the store. The database will have all the data related to the medical 
store, this includes all the medicines available, details of the employees and details of the customers.
	The resulting web-application template created will help in better management and organization 
of data.
